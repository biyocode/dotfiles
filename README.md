# Dotfiles

## Current Machine: Arch
 - Shell: bash
   - Prompt: Starship
 - WM: bspwm
 - Greeter: lightdm
 - Terminal: kitty
 - Editor: neovim
 - Status Bar: polybar
 - Compositor: picom
 - Menu: rofi
 - File Explorer: thunar-git
 - Search: fzf, fd, ripgrep

## Themes

Most of my themes are custom colors by me.

 - vim-airline: navarch (nvim/mythemes/airline -> nvim/plugged/vim-airline/autoload/airline/themes)
